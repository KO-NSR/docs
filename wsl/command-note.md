# 🧰 **WSL コマンド集**

## 🚀 **基本操作**
| コマンド | 説明 |
|---------|------|
| `wsl` | 既定のディストリビューションを起動 |
| `wsl -d <名前>` | 指定したディストリビューションを起動 |
| `wsl --help` | コマンド一覧を表示 |

---
## 🖥 **起動・終了・状態確認**
| コマンド | 説明 |
|---------|------|
| `wsl --status` | WSL の状態を確認 |
| `wsl --shutdown` | すべての WSL を停止 |
| `wsl -t <ディストリ名>` | 指定ディストリを停止 |

---

## 📦 **ディストリビューション管理**
| コマンド | 説明 |
|---------|------|
| `wsl --list` | インストール済みディストリビューション一覧 |
| `wsl --list --verbose` | 状態・バージョン付きで一覧表示 |
| `wsl --list --online` | インストール可能なディストリビューション一覧 |
| `wsl --install` | WSL と Ubuntu をまとめてインストール |
| `wsl --install <ディストリ名>` | 指定ディストリをインストール |
| `wsl --set-default <ディストリ名>` | 既定ディストリを変更 |
| `wsl --unregister <ディストリ名>` | ディストリを削除（初期化） |

---

## 🔧 **WSL バージョン管理**
| コマンド | 説明 |
|---------|------|
| `wsl --set-version <ディストリ名> 2` | WSL2 に変更 |
| `wsl --set-version <ディストリ名> 1` | WSL1 に変更 |
| `wsl --set-default-version 2` | 新規インストールを WSL2 に設定 |

---


## 📁 **ファイル操作（Windows ↔ Linux）**
| コマンド | 説明 |
|---------|------|
| `explorer.exe .` | Linux から Windows エクスプローラーを開く |
| `/mnt/c/...` | Windows の C ドライブへアクセス |
| `wslpath <パス>` | Windows ↔ Linux のパス変換 |

---

## 🌐 **ネットワーク関連**
| コマンド | 説明 |
|---------|------|
| `ip addr` | Linux 側の IP を確認 |
| `cat /etc/resolv.conf` | DNS 設定を確認 |

---

## 📦 **バックアップ・移行**
| コマンド | 説明 |
|---------|------|
| `wsl --export <ディストリ名> <tarファイル>` | ディストリをエクスポート |
| `wsl --import <名前> <フォルダ> <tarファイル>` | ディストリをインポート |

---

## 🧪 **よく使う実用コマンド例**
- `wsl -u root` → root で起動  
- `wsl --exec <コマンド>` → Windows から Linux コマンド実行  
- `wsl --mount <デバイス>` → 物理ディスクをマウント（WSL2）
